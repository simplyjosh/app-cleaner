## Setup guide

### Step 1: Preperation

```shell
sudo mkdir -p /usr/local/bin
sudo chown ${USER}:staff /usr/local/bin
```

### Step 2: Download & Install

```shell
curl -o /usr/local/bin/app-cleaner.sh 
chmod +x /usr/local/bin/app-cleaner.sh
```

## Usage guide

```console
$ app-cleaner.sh /Applications/AppName.app
```
